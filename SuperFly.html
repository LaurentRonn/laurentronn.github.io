SuperFly: a Oculus Go game

A pick-up flying game for the Oculus Go.

Well... This was supposed to be released in October for the Oculus Go platform, however with the recently anouncements made by Facebook to discard the Oculus Go platform, seems like I'll just upload it to GitHub.
Or maybe rework it and switch platform to Oculus Quest.

In any case, this game is a simple exercise in design, and has been a very interesting challenge: building a interactive experience for VR with the Oculus Go using a single controller is like fighting with one hand behind your back.
